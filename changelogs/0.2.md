# v0.2

## Features

### Forge

- Added a Forge installer (and uninstaller too)
- Select your instance, go to `Mods` -> `Install Forge` and you're done!
- Note: Only release 1.6.4 and above are currently supported.

### Logs

- Now you can view game logs!
- When the game launches, the logs will be available on the right side.
- Only a section of the log is shown for performance reasons
- Press `Copy Logs` to copy the **full** logs (if you face any issues you can send it)

### Misc

- Added an update detector that looks for launcher updates and asks to install it.
  - If you accept, it automatically updates the launcher and create a backup for you.
- The UI design has been changed to be cleaner.
- Added progress bar for Fabric installer.
- Added uninstaller for Fabric.
- Now there are basic command line options: `--version` and `--help`
  - (Proper headless CLI mode added in v0.4.2)
- Game current directory is now the `.minecraft` folder. Fixes random files popping up near the launcher executable.
- Debug launcher logs now incorporate colored text and dynamic progress bars.
